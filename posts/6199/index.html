REST API 介绍

本接口规范对XXX小程序接口操作，相关参数、响应和错误码进行定义，所有提交及返回接收的变量均使用小写。目前仅提供 HTTP(S) 接口，并提供接口操作请求和响应范例。

调用方法

请求URL

REST API 的 URL 格式如下：

https://test.easyboot.xyz/$ver/$servicename/$command?sign=$sign&random=$random&timestamp=$timestamp

其中各个参数的含义以及取值如下（参数名称及其取值均区分大小写）：

参数 含义 取值
https 请求协议 请求协议为 HTTPS，请求方式为 POST
cxjwx.saicdh.com 请求域名 固定为 test.easyboot.xyz
ver 版本号 固定为 v1
servicename 内部服务名 示例：v1/user/login，其中user为servicename
command 命令字，与 servicename 组合用来标识具体的业务功能 示例：v1/user/login，其中login为command
sign 签名 根据body中参数进行加密签名
random 随机数 32位无符号整数随机数
timestamp 时间戳 请求时间转换成的10位时间戳

Sign签名方式

  1. 将Body的JSON参数用ASCII码进行排序，并通过&连接，如：a=a&b=b&c=c
  2. 对排序后的参数进行MD5加密，加密后的字符串即为sign
  3. 把生成的sign追加在请求URL后面，如：https://xxx.com?sign=$sign

HTTP 请求包体格式

REST API 仅支持 POST 方法，其请求包体为 JSON 格式，具体的包体格式参见每个 API 的详细描述。
需要特别注意的是，POST 包体不能为空，即使某条协议包体中不需要携带任何信息，也需要携带一个空的 JSON 对象，即 {}。

HTTP 返回码

除非发生网络错误（例如502错误），否则 REST API 的调用结果均为200，真正的 API 调用错误码与错误信息在 HTTP 应答包体中返回。

HTTP 应答包体格式

REST API 的应答包体也是 JSON 格式，其格式符合如下特征：

{
    "code": 200, 
    "msg": "success", 
    "data": {}
}

应答包体中必然包含 code、msg、data 这三个属性，其含义如下：

字段 类型 说明
code int 错误码，200为成功，其他为失败
msg String 失败原因
data Object 成功后返回的数据

请求示例

以下为通过 REST API来进行用户登录操作的示例。
HTTPS 请求：

POST /v1/user/login?sign=xxx&random=99999999&timestamp=1546335259 
Host: test.easyboot.xyz
Body: {
    "username": "test",
    "password": "test_pass"
}

HTTPS 应答：

{
    "code": 200, 
    "msg": "操作成功", 
    "data": {
         "uid": 123456,
         "nickName": "test user",
         "avatar": "https://xxx.com/xxx.jpg"
    }
}

Java校验源码

import cn.hutool.core.date.DateUtil;
import cn.hutool.core.util.ReUtil;
import cn.hutool.core.util.StrUtil;
import cn.hutool.crypto.SecureUtil;
import cn.hutool.extra.servlet.ServletUtil;
import cn.hutool.json.JSONUtil;
import org.apache.shiro.web.filter.AccessControlFilter;

import javax.servlet.ServletRequest;
import javax.servlet.ServletResponse;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
import java.io.IOException;
import java.io.PrintWriter;
import java.util.Arrays;
import java.util.Enumeration;
import java.util.Map;
import java.util.Set;
public static final String TIMESTAMP = "timestamp";
public static final String RANDOM = "random";
public static final String SIGN = "sign";
HttpServletRequest request = (HttpServletRequest) servletRequest;
       HttpServletResponse response = (HttpServletResponse) servletResponse;

// 请求是否过期
       String timestampStr = request.getParameter(TIMESTAMP);
       if (StrUtil.isBlank(timestampStr)) {
           renderError(response, "缺少参数：timestamp");
           return false;
       }
       long timestamp = Long.parseLong(timestampStr);
       long currDate = DateUtil.currentSeconds();
       if (timestamp + 18000 < currDate) {
           renderError(response, "timestamp已过期");
           return false;
       }

       // 检查随机数
       String pattern = "[1-9]\\d*";
       String random = request.getParameter(RANDOM);
       if (StrUtil.isBlank(random)) {
           renderError(response, "缺少参数：random");
           return false;
       } else if (!(ReUtil.isMatch(pattern, random) && random.length() == 32)){
           renderError(response, "random非法");
           return false;
       }

       // 检查sign
       String signature = request.getParameter(SIGN);
       if (StrUtil.isBlank(signature)) {
           renderError(response, "缺少参数：sign");
           return false;
       }

       // 请求参数
       String body = ServletUtil.getBody(request);
       if (StrUtil.isBlank(body)) {
           renderError(response, "请求体body不能为空");
           return false;
       }
       Map map = JSONUtil.toBean(body, Map.class);
       // 参数排序
       Set keySet = map.keySet();
       Object[] keys = keySet.toArray();
       Arrays.sort(keys);
       StringBuilder strBuilder = new StringBuilder();
       for (int i = 0; i < keys.length; i++) {
           Object str = keys[i];
           strBuilder.append(str.toString())
                   .append("=")
                   .append(map.get(str.toString()));
           if (i != keys.length - 1) {
               strBuilder.append("&");
           }
       }

       // 校验sign
       String newSignature = SecureUtil.md5(strBuilder.toString());
       if (!StrUtil.equals(signature, newSignature)) {
           renderError(response, "sign校验失败");
           return false;
       }
private void renderError(HttpServletResponse response, String message) {
       if (StrUtil.isEmpty(message)) {
           message = "非法请求";
       }

       response.setContentType("application/json;charset=UTF-8");
       PrintWriter out = null;
       try {
           out = response.getWriter();
           out.write("{\"message\":\""+ message +"\",\"code\":403}");
           out.flush();
       } catch (IOException e) {
           e.printStackTrace();
       }
}
